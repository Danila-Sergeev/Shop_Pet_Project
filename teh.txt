стек технологий:
1. Node.js - платформа для backend на js
2. Express - фреймворк для написания beckend на node.js
3. Postgr.sql - система управления БД
4. Sequelize - ORM для РБД на node.js (ORM связывает программный код с БД)
5. cors - для обращения с браузера к серверу 
6. nodemon - для автоматического перезапуска сервера при обновлении
7. JWT (json web token)
8. bcrypt - хеширует пароли
